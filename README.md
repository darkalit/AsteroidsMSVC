# AsteroidsMSVC
This game was made just for practicing in C++. Just copied gameplay of Asteroids on Atari

![AsteroidsMSVCimage1](/Screenshots/scrn1.png)


## Controls:
- Z, X - Shoot;
- Up - Go forward;
- Left, Right - Rotate the ship;
- P - Pause;
- Esc - Exit.
